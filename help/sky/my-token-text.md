# Mein Token: Text

<br> 

So richten Sie ein Text-Token für die Verwendung in Ihren Programmen und Kampagnen-Ordnern ein.

1. Ziehen Sie auf der Registerkarte &quot; [!UICONTROL Meine Token] &quot;Ihres Programms oder Ordners ein **Text** -Token auf die Arbeitsfläche &quot; [!UICONTROL Lokale Token] &quot;.

   ![Bild eins](/help/sky/assets/my-tokens/my-token-text/my-token-text-1.png)

1. Geben Sie einen Namen für das neue Token ein und fügen Sie dann den gewünschten Text hinzu. Klicken Sie auf [!UICONTROL **Speichern**].

   ![Bild zwei](/help/sky/assets/my-tokens/my-token-text/my-token-text-2.png)
