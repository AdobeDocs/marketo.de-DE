## Ankündigungen {#announcements}

Die Echtzeit-Personalisierung von Marketo Engage wurde um zwei Verbesserungen erweitert.

* **Upgrade auf Google Analytics 4**: RTP-Benutzer verfügen über ein eindeutiges alphanumerisches Tag (im `<head>` -Abschnitt des HTML-Codes) auf ihren Landingpages, über die Marketo Daten an Google Analytics senden kann. Marketo hat zuvor Universal Analytics verwendet, das die Aufzeichnung von Daten bis zum 1. Juli 2023 stoppt. RTP-Benutzer müssen den Migrationsprozess in ihrem Google Analytics-Dashboard ausführen, um die Migration abzuschließen.

* **Aktualisierung auf jQuery 3.7**: Um die Sicherheit und Stabilität zu erhöhen, aktualisiert Marketo auf jQuery 3.7.0. RTP-Benutzer, die jQuery allein laden, können sich negativ auswirken, wenn sie eine Version verwenden, die älter als 3.0.0 ist. Diese Benutzer können sich an den Marketo-Support wenden, um Hilfe bei der Implementierung einer bestimmten Version von rtp.js zu erhalten, die ihren Anforderungen entspricht. Wenn Sie Version 3.0.0 oder höher verwenden, sind Sie davon nicht betroffen.
