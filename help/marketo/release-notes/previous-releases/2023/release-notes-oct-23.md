# Versionshinweise – Oktober 2023 {#release-notes-oct-23}

Im Folgenden finden Sie alle Funktionen, die in der Version vom Oktober 2023 enthalten sind. Überprüfen Sie Ihre Adobe Marketo Engage-Edition auf die Verfügbarkeit der Funktionen.

>[!AVAILABILITY]
>
>Mit einem Stern (![star](assets/yellow-star.png) gekennzeichnete Funktionen sind kostenpflichtige Add-ons. Weitere Informationen erhalten Sie vom Marketo Engage-Support.

Oktober 2023** veröffentlicht, wobei die verbleibenden Funktionen in den folgenden Wochen schrittweise bereitgestellt werden. Veröffentlichungsfunktionen und -daten können sich ändern. ## Ankündigungen {#announcements}

* **Marketo Engage Forms jQuery 1.x**: In unserer Version vom Januar 2024 aktualisieren wir die jQuery für Marketo Engage Forms auf jQuery 3.x. Dies kann sich auf die Implementierung benutzerdefinierter Formulare auswirken, die auf älteren Versionen von jQuery basieren. [Weitere Informationen finden Sie hier](https://nation.marketo.com/t5/product-blogs/marketo-engage-forms-amp-forms2-js-jquery-update/ba-p/341705#M2597){target="_blank"}.

* **Einstellung von Marketo Events und Marketo Moments**: Adobe entfernt [Marketo Events und Marketo Moments](https://nation.marketo.com/t5/product-discussions/marketo-events-app-and-marketo-moments-app-end-of-life/m-p/340712/highlight/true#M193869){target="_blank"} ab dem 2. Oktober 2023 aus allen App-Stores. Vorhandene Benutzende: Sie können diese Apps weiterhin verwenden, bis Sie Ihre Marketo Engage-Instanz zu Adobe Identity migrieren. Diese Apps unterstützen nicht die Anmeldung bei Adobe ID.
