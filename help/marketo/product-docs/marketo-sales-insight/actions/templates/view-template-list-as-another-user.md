# Vorlagenliste als einen anderen Benutzer anzeigen {#view-template-list-as-another-user}

Als Administrator können Sie Vorlagen wie beliebige Benutzer anzeigen.

>[!NOTE]
>
>**Administratorberechtigungen erforderlich**

1. Klicken Sie auf **[!UICONTROL Vorlagen]**.

   ![](assets/view-template-list-as-another-user-1.png)

1. Klicken Sie auf **[!UICONTROL Anzeigen als]** und wählen Sie den gewünschten Benutzer aus.

   ![](assets/view-template-list-as-another-user-2.png)

1. Sie zeigen jetzt Vorlagen als ausgewählten Benutzer an.

   ![](assets/view-template-list-as-another-user-3.png)

   >[!NOTE]
   >
   >Sie können auch Filter oder die Suchfunktion zusammen mit &quot;_[!UICONTROL anzeigen als]_ verwenden, um anzuzeigen, was für Sie am relevantesten ist.
