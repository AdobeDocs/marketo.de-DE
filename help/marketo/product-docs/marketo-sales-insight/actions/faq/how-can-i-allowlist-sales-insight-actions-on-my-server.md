# Auf die Zulassungsliste setzen Wie kann ich [!DNL Sales Insight Actions] auf meinem Server speichern? {#how-can-i-allowlist-sales-insight-actions-on-my-server}

Leider gibt es keine Möglichkeit, eine statische IP-Adresse bereitzustellen. [!DNL Sales Insight Actions] wird in der Cloud bereitgestellt und verwendet über 200 Server, um E-Mails und Benutzeranfragen zu verarbeiten. Und eine sehr große Reichweite zu bieten, wäre nicht sicherer.
