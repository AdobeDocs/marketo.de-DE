# Web-Aktivitäten {#web-activities}

Die Registerkarte Web-Aktivitäten zeigt die Web-Aktivitäten Ihrer Leads/Kontakte an.
Überprüfen Sie die neuesten Web-Aktivitäten Ihres Leads unter Angabe der Anzahl der Seitenbesuche und der entsprechenden Konten. Sie können die Ergebnisse filtern, um die angegebene Anzahl von Seiten zu begrenzen.

![](assets/web-activities-1.png)

## Anonyme Web-Aktivitäten {#anonymous-web-activities}

Auf der Registerkarte Anonyme Web-Aktivitäten werden alle **anonymen** Web-Seiten-Besucheraktivitäten angezeigt. Überprüfen Sie die neuesten Web-Aktivitäten unter Angabe der Anzahl der Seitenbesuche.
