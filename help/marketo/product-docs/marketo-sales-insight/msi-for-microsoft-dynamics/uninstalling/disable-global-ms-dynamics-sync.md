# Globale MS Dynamics-Synchronisierung deaktivieren {#disable-global-ms-dynamics-sync}

Führen Sie diese einfachen Schritte aus, um die MS Dynamics Synchronisierung zu deaktivieren.

1. Klicken Sie in Marketo auf **Admin**.

   ![](assets/one.png)

1. Klicken Sie unter Integration auf **Microsoft Dynamics**.

   ![](assets/two.png)

1. Klicken Sie auf **Synchronisierung deaktivieren**.

   ![](assets/three.png)

   >[!NOTE]
   >
   >Wenn in Ihrer Instanz keine Schaltfläche zum Deaktivieren der Synchronisierung angezeigt wird, wenden Sie sich bitte an [MarketingToSupport](https://nation.marketo.com/t5/Support/ct-p/Support).
