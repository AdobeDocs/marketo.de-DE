# Suchen von Web-Kampagnen, die ein bestimmtes Segment verwenden {#find-web-campaigns-that-are-using-a-specific-segment}

Suchen Sie nach Web-Kampagnen, die ein bestimmtes Segment verwenden?

1. Gehen Sie zu **Segmente**.\
   ![](assets/new-dropdown-segments-hand-1.jpg)

1. Suchen Sie nach einem **Segment**. Wählen Sie den **Segmentnamen** aus. Klicken Sie im rechten Fensterbereich auf **Zugeordnete Kampagnen** , um die Kampagnen, die diesem bestimmten Segment zugeordnet sind, Ansicht.\
   ![](assets/image2014-11-26-14-21-59.png)

1. Ansicht der **Kampagnen** , die mit dem ausgewählten Segment verbunden sind.\
   ![](assets/image2014-11-26-14-3a25-3a30.png)
