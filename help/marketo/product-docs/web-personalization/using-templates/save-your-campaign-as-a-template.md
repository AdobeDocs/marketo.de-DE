# Speichern Ihrer Kampagne als Vorlage {#save-your-campaign-as-a-template}

Haben Sie schon einmal die perfekte Web-Kampagne entwickelt? Sie können sie jetzt als Vorlage speichern, um sie in Zukunft einfach wiederzuverwenden.

1. Gehen Sie zu Web-Kampagnen.

   ![](assets/web-campaigns-hand-1.jpg)

1. Suchen Sie nach der Kampagne, die Sie als Vorlage speichern möchten.

   ![](assets/search-for-campaign.jpg)

1. Klicken Sie auf das Symbol Bearbeiten.

   ![](assets/my-campaign-edit.jpg)

1. Aktivieren Sie Als Vorlage verwenden und klicken Sie auf Speichern.

   ![](assets/image2015-2-25-19-3a56-3a58.png)   ![](assets/image2015-2-25-19-3a56-3a37.png)

1. Das ist&#39;s! Wenn Sie das nächste Mal eine Kampagne erstellen und eine Vorlage auswählen, aktivieren Sie auf der Seite &quot;Kampagnen festlegen&quot;die Option &quot;Meine Vorlagen&quot;, um die gespeicherten Vorlagen anzuzeigen.

   ![](assets/image2014-9-17-20-3a55-3a31.png)
