# Berichtanmeldungen aktivieren/deaktivieren {#enable-disable-report-subscriptions}

Web Personalization verfügt über eine Reihe nützlicher Berichte, die per E-Mail versendet werden. Hier ist, wie man sie abonniert.

1. Melden Sie sich bei Web Personalization an. Klicken Sie unter Ihrem Anmeldenamen auf **Benutzereinstellungen**.

   ![](assets/image2014-9-17-20-3a48-3a28.png)

1. Wählen Sie den Bericht aus, den Sie abonnieren möchten, und geben Sie die Häufigkeit an. Klicken Sie auf **Speichern**.

   ![](assets/email-settings.png)

   Das ist es! Um sich vom Bericht abzumelden, deaktivieren Sie einfach das Kontrollkästchen und klicken Sie auf **Speichern**.
