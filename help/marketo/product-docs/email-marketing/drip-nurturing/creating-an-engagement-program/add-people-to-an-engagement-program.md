# hinzufügen Personen zu einem Interaktions-Programm {#add-people-to-an-engagement-program}

Es gibt mehrere Möglichkeiten, wie Menschen Mitglieder eines Interaktionskonzepts werden können:

* [Programm Hinzufügen Interaktion](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/add-to-engagement-program.md)
* [Schritt zum Ändern des Interaktions-Programm ](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/change-engagement-program-cadence.md) Cadenceflow
* [Schritt zum Ändern des Interaktions-Programm-](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/change-engagement-program-stream.md) Streamflows
* [Schritt ](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/program-flow-actions/change-program-status.md) zum Ändern des Status des Programms
* Wenn eine Person ein Formular lokal in ein Interaktions-Programm ausfüllt
* Importieren von Mitgliedern in ein Interaktions-Programm (über die Registerkarte &quot;Mitglieder&quot;innerhalb des Programms)
* Wenn Sie Salesforce als CRM verwenden und eine Salesforce-Kampagne an Ihr Interaktions-Programm gebunden ist, werden Sie durch das Hinzufügen einer Person zur Salesforce-Kampagne auf der CRM-Seite ebenfalls zum Einsatz-Programm.
