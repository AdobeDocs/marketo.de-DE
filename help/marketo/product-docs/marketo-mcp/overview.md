# MCP-Übersichtsdokument {#overview}

TEXT DES VATERS:

Der öffentliche Marketo Engage-MCP-Server ist ein Cloud-basierter Service, der Marketo-REST-APIs als strukturierte Tools für KI-Agenten zugänglich macht. Sie basiert auf dem Model Context Protocol (MCP) und bietet eine standardisierte, schemabasierte Schnittstelle, über die KI-Systeme sicher und effizient mit Marketo interagieren können.

Anstatt Marketo-Endpunkte manuell in jede Anwendung zu integrieren, stellt der MCP-Server ein einziges, KI-natives Gateway bereit, mit dem MCP-kompatible Agenten verfügbare Vorgänge ermitteln, erforderliche Parameter verstehen und Aktionen über ein konsistentes Protokoll ausführen können.

Es gibt eine Vielzahl von Tools, die sowohl öffentliche REST-APIs als auch interne APIs umfassen und die Marketo-Kernfunktionen wie in der Benutzeroberfläche, aber über API-Aufrufe, ausführen. Weitere Tools werden im Zuge der Entwicklung laufender Tools hinzugefügt. &lt;---- ERFORDERT KLARSTELLUNG

Der MCP-Server wird als Self-Service angeboten und kann über das Adobe IO-Framework eingerichtet werden, indem ein Endpunkt erstellt und die Verbindung authentifiziert wird. Weitere Informationen und Details zur Einrichtung finden Sie auf der Adobe Developer-Website (Link hier hinzufügen). DIESER? https://developer.adobe.com/marketo-apis/

Campaign-Tools, die als SOAP-APIs angezeigt werden, wurden in REST-APIs konvertiert und hinzugefügt.

Um eine Liste der verfügbaren Tools anzuzeigen, folgen Sie nach der Konfiguration diesen Anweisungen hier (Hyperlink).

HINWEIS: Nicht verfügbare Tools sind „Kampagne deaktivieren“ und „Statische Liste löschen“ (TBD).
