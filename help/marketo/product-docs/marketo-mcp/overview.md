# MCP-Übersichtsdokument {#overview}

TEXT DES VATERS:

Der öffentliche MCP-Server für Marketo Engage ist ein Cloud-zugänglicher Service, der die volle Leistungsfähigkeit der öffentlichen Marketo Engage-REST-APIs als strukturierte, auffindbare Tools für KI-Agenten verfügbar macht. Es baut auf dem Model Context Protocol (MCP) auf und verwandelt die Marketing-Automatisierungsfunktionen von Marketo in eine standardisierte, schemabasierte Benutzeroberfläche, die KI-Systeme sicher und intelligent verwenden können.

Anstatt Marketo-Endpunkte manuell in jede Anwendung zu integrieren, fungiert der MCP-Server als ein einziges, natives Gateway, sodass MCP-kompatible Agenten verfügbare Vorgänge ermitteln, erforderliche Parameter verstehen und Aktionen über ein konsistentes Protokoll ausführen können.

Es gibt eine Vielzahl von Tools, die sowohl öffentliche REST-APIs als auch interne APIs umfassen und die Marketo-Kernfunktionen wie in der Benutzeroberfläche, aber über API-Aufrufe, ausführen. Weitere Tools werden im Zuge der Entwicklung laufender Tools hinzugefügt.

Der MCP-Server wird als Self-Service angeboten und kann über das Adobe IO-Framework eingerichtet werden, indem ein Endpunkt erstellt und die Verbindung authentifiziert wird. Weitere Informationen und Details zur Einrichtung finden Sie auf der Adobe Developer-Website unter (Link hier hinzufügen).

Campaign-Tools, die als SOAP-APIs angezeigt werden, wurden in REST-APIs konvertiert und hinzugefügt.
