# In-App-Nachricht senden {#send-your-in-app-message}

Nachdem Sie Ihre In-App-Nachricht erstellt haben, ist es Zeit, sie zu versenden. Gehen Sie wie folgt vor:

>[!NOTE]
>
>Wenn Sie eine von App Open ausgelöste In-App-Nachricht testen, müssen Sie die App auf das Telefon herunterladen, öffnen und schließen. Dadurch wird der SDK-Code initialisiert. Wenn Sie die App das nächste Mal öffnen, wird die In-App-Nachricht angezeigt. Benutzerdefinierte Ereignisse werden beim ersten Öffnen der App Trigger.

1. Klicken Sie in Marketingaktivitäten auf den Ordner, in dem das Programm live geschaltet werden soll, und wählen Sie **Neues Programm** aus.

   ![Bild 1](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/assets/send-your-in-app-message-1.png)

1. Geben Sie einen Namen ein.

   ![Bild 2](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/assets/send-your-in-app-message-2.png)

1. Wählen Sie aus der Dropdown-Liste den In-App-Programmtyp aus und klicken Sie auf **Erstellen** .

   ![Bild 3](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/assets/send-your-in-app-message-3.png)

1. Verwenden Sie das Control Panel, um festzulegen, wer, was und wann der In-App-Nachricht gesendet wird. Wenn Sie damit fertig sind, genehmigen Sie sie. Für jeden Schritt gibt es einen eigenen Artikel.

   ![Bild Vier](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/assets/send-your-in-app-message-4.png)

Bereit? Beginnen Sie mit [Festlegen der Audience](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/set-your-in-app-message-audience.md).
