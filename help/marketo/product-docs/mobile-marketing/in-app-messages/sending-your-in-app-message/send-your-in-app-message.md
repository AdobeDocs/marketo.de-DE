# In-App-Nachricht senden {#send-your-in-app-message}

Nachdem Sie Ihre In-App-Nachricht erstellt haben, können Sie sie jetzt versenden. Im Folgenden finden Sie die Schritte:

>[!NOTE]
>
>Wenn Sie eine In-App-Nachricht testen, die von [!UICONTROL App Open] ausgelöst wird, laden Sie die App auf das Smartphone herunter, öffnen Sie sie und schließen Sie sie. Dadurch wird der SDK-Code initialisiert. Beim nächsten Öffnen der App wird die In-App-Nachricht angezeigt. Trigger bei benutzerspezifischen Ereignissen beim ersten Öffnen der App.

1. Klicken [!UICONTROL  unter &quot;]&quot; auf den Ordner, in dem das Programm gespeichert werden soll, und wählen Sie **[!UICONTROL Neues Programm]**.

   ![Bild eins](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/assets/send-your-in-app-message-1.png)

1. Geben Sie einen Namen ein.

   ![Bild zwei](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/assets/send-your-in-app-message-2.png)

1. Wählen Sie aus der Dropdown-Liste den In-App-Programmtyp aus und klicken Sie auf **[!UICONTROL Erstellen]**.

   ![Bild 3](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/assets/send-your-in-app-message-3.png)

1. Verwenden Sie [!UICONTROL Control Panel] um festzulegen, wer, was und wann die In-App-Nachricht sendet und wann Sie damit fertig sind, genehmigen Sie sie. Für jeden Schritt gibt es einen separaten Artikel.

   ![Bild vier](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/assets/send-your-in-app-message-4.png)

Bereit? Beginnen Sie mit [Festlegen der Zielgruppe](/help/marketo/product-docs/mobile-marketing/in-app-messages/sending-your-in-app-message/set-your-in-app-message-audience.md).
