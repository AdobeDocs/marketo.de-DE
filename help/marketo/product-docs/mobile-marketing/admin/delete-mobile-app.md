# Mobile App {#delete-mobile-app} löschen

1. Klicken Sie auf **Admin**.

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. Wählen Sie **Mobile Apps**.

   ![](assets/image2015-4-22-16-3a14-3a29.png)

1. Wählen Sie die gewünschte mobile App aus.

   ![](assets/image2015-4-22-17-3a22-3a11.png)

1. Klicken Sie auf **Mobile App Actions** und wählen Sie **App** löschen.

   ![](assets/image2015-4-22-17-3a21-3a51.png)

1. Bestätigen Sie dies, indem Sie auf **Löschen** klicken.

   ![](assets/image2015-4-22-17-3a23-3a18.png)

Ta-da! Push-Benachrichtigungen können nicht mehr von dieser mobilen App gesendet werden.
