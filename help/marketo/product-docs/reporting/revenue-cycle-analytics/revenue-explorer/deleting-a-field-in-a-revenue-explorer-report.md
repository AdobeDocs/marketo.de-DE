# Löschen eines Felds in einem [!UICONTROL Umsatz-Explorer]-Bericht {#deleting-a-field-in-a-revenue-explorer-report}

Manchmal kann es vorkommen, dass Sie ein falsches Feld in den Bericht ziehen. So entfernen Sie es:

1. Ziehen Sie das Feld, das Sie entfernen möchten, auf das **Papierkorb**-Symbol unten rechts im Bericht.

   ![](assets/image2015-3-24-16-3a40-3a13.png)

   >[!NOTE]
   >
   >Der Papierkorb wird ausgeblendet, bis Sie ein Feld ziehen.
