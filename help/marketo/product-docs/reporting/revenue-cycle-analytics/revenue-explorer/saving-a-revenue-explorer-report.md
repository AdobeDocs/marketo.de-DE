# Speichern eines Umsatz-Explorer-Berichts {#saving-a-revenue-explorer-report}

Umsatz-Explorer-Berichte können in der gewünschten Datei gespeichert werden.

1. Klicken Sie auf das Symbol Speichern .

   ![](assets/image2015-3-25-17-3a8-3a49.png)

   >[!NOTE]
   >
   >Änderungen, die Sie an Ihrem Bericht vornehmen, werden nicht automatisch gespeichert. Also achten Sie darauf, oft zu sparen!

1. Geben Sie Ihrem Bericht einen beschreibenden Namen, wählen Sie einen Ort aus und klicken Sie auf **Speichern**!

   ![](assets/image2015-3-26-13-3a30-3a33.png)

   Das ist alles! Sie können jetzt auf Ihre Datei in **Dateien durchsuchen**.

   ![](assets/image2015-3-27-11-3a32-3a51.png)
