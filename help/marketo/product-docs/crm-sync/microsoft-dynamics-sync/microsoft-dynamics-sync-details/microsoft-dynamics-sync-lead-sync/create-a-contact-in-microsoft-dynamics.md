# Erstellen eines Kontakts in Microsoft Dynamics {#create-a-contact-in-microsoft-dynamics}

1. Wählen Sie die einzige Person für Marketo (Microsoft Type is empty) aus, die Sie als Kontakt in Dynamics erstellen möchten.

   ![](assets/one.png)

1. Klicken **Personenaktionen** und **Microsoft** und wählen Sie **Person mit Microsoft synchronisieren**.

   ![](assets/two.png)

1. Klicken **Synchronisieren als** und wählen Sie **Kontakt**. Klicken **Jetzt ausführen**.

   ![](assets/three.png)

   >[!NOTE]
   >
   >Bei Verwendung der FlusAktion &quot;Person mit Microsoft synchronisieren&quot;(nur in einer Trigger-Kampagne) wird der Lead/Kontakt in Echtzeit in Dynamics erstellt.

1. Marketo qualifiziert diesen Lead-Datensatz in Dynamics in einen Kontakt, der keinem Konto in Dynamics zugeordnet ist.

   ![](assets/image2015-10-23-9-3a43-3a33.png)

1. Jetzt können Sie **Kontakt** wenn Sie die Einschränkung &quot;Synchronisieren als&quot;in einem Smart-Kampagnenfilter verwenden.

   ![](assets/five.png)
