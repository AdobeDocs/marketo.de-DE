# Microsoft Dynamics Sync: Opportunity Sync {#microsoft-dynamics-sync-opportunity-sync}

Marketo to Dynamics sync ist super leistungsfähig. Im Folgenden finden Sie alle Details zur Opportunitätssynchronisierung:

## Wie werden Opportunitätsdetails zwischen den beiden Systemen synchronisiert? {#how-are-opportunity-details-kept-in-sync-between-the-two-systems}

Die Opportunitätssynchronisierung ist eine Möglichkeit - Dynamik nach Marketo. Wenn Sie Änderungen an einer Gelegenheit in Dynamics vornehmen, wird Ihr Update in Marketo angezeigt.

## Kann ich mit Marketo eine Chance in Dynamics schaffen? {#can-i-create-an-opportunity-in-dynamics-using-marketo}

Nein, Sie müssen die Chance in Dynamics schaffen und es wird automatisch mit Marketo synchronisiert.

## Welche Felder werden mit Marketo synchronisiert? {#what-fields-will-sync-to-marketo}

Sie können die zu synchronisierenden Felder [während des Setups auswählen.](/help/marketo/product-docs/crm-sync/microsoft-dynamics-sync/sync-setup/microsoft-dynamics-365/step-3-of-3-connect.md#select-fields-to-sync)
