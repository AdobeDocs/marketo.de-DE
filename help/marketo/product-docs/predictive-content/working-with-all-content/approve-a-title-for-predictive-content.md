# Genehmigen eines Titels für Prognoseinhalte {#approve-a-title-for-predictive-content}

Sie können einen beliebigen Titel auf der Seite „Alle Inhalte“ zu prädiktiven Inhalten hinzufügen, indem Sie ihn auf der Seite „Alle Inhalte“ oder im Popup „Inhalt bearbeiten“ genehmigen.

## Gesamte Inhaltsseite {#all-content-page}

1. Aktivieren Sie das Kontrollkästchen neben dem Inhalt.

   ![](assets/image2017-10-3-9-3a9-3a47.png)

1. Klicken Sie auf **Inhaltsaktionen** und wählen Sie **Für prädiktiven Inhalt genehmigen** aus.

   ![](assets/image2017-10-3-9-3a10-3a31.png)

## Popup „Inhalt bearbeiten“ {#edit-content-pop-up}

Sie können Titel für prädiktive Inhalte auch direkt im Popup Inhalt bearbeiten genehmigen.

1. Bewegen Sie den Mauszeiger über ein Inhaltselement und klicken Sie auf das Bearbeitungssymbol am Ende der Zeile.

   ![](assets/image2017-10-3-9-3a14-3a55.png)

1. Aktivieren Sie das **Für prädiktiven Inhalt genehmigen** im Popup Inhalt bearbeiten und klicken Sie auf **Speichern**.

   ![](assets/image2017-10-3-9-3a15-3a35.png)

Unabhängig davon, welche Methode Sie verwenden, wird das Symbol Für prädiktiven Inhalt genehmigen jetzt in der Zeile angezeigt.

![](assets/five.png)

Jetzt können Sie den Titel auf der Seite für prädiktive Inhalte sehen.

![](assets/image2017-10-3-9-3a16-3a45.png)
