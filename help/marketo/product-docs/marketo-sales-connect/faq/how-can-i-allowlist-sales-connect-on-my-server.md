# Auf die Zulassungsliste setzen Wie kann ich eine Verbindung zum Vertrieb auf meinem Server herstellen? {#how-can-i-allowlist-sales-connect-on-my-server}

Leider gibt es keine Möglichkeit, eine statische IP-Adresse bereitzustellen. Sales Connect wird in der Cloud bereitgestellt und nutzt über 200 Server zur Verarbeitung von E-Mails und Benutzeranfragen. Und ein sehr umfangreiches Sortiment zu bieten, wäre für Sie nicht sicherer.
