# Auf die Zulassungsliste setzen Wie kann ich [!DNL Sales Connect] auf meinem Server speichern? {#how-can-i-allowlist-sales-connect-on-my-server}

Leider gibt es keine Möglichkeit, eine statische IP-Adresse bereitzustellen. [!DNL Sales Connect] wird in der Cloud bereitgestellt und verwendet über 200 Server, um E-Mails und Benutzeranfragen zu verarbeiten. Und ein sehr umfangreiches Sortiment zu bieten, wäre für Sie nicht sicherer.
