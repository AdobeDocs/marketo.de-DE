# Hinzufügen von Hyperlink-Text {#add-hyperlinked-text}

1. Geben Sie den Text ein, für den Sie einen Hyperlink erstellen möchten (z. B. „Hier klicken„).

1. Wählen Sie den Text aus.

1. Klicken Sie auf die Schaltfläche Link im Editor.

1. Geben Sie die URL ein, mit der sie verknüpft werden soll (z. B. &quot;<https://www.mywebsite.com>„).

1. Klicken Sie auf **[!UICONTROL OK]**.
