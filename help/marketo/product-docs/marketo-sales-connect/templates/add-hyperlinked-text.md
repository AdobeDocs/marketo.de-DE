# hinzufügen Hyperlink-Text {#add-hyperlinked-text}

1. Geben Sie den Text ein, den Sie per Hyperlink verknüpfen möchten (z. B. hier klicken).
1. Wählen Sie den Text aus.
1. Klicken Sie im Editor auf die Verknüpfungsschaltfläche.
1. Geben Sie die URL ein, mit der Sie eine Verknüpfung herstellen möchten (z. B. http://www.mywebsite.com).
1. Klicken Sie auf **OK**.
