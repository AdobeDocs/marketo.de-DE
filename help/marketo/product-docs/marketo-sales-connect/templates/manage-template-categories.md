# Verwalten von Vorlagenkategorien {#manage-template-categories}

## Erstellen einer neuen Kategorie {#create-a-new-category}

[!DNL Sales Connect] verfügt über zwei Standardkategorien, um Ihre Vorlagen zu organisieren.

1. Klicken Sie auf das Symbol **+** neben **[!UICONTROL Kategorien]**.

1. Geben Sie einen Namen für Ihre neue Kategorie ein und klicken Sie dann auf **[!UICONTROL Erstellen]**.

Die neue Kategorie wird jetzt im Navigationsmenü angezeigt.

## Bearbeiten einer Vorlagenkategorie {#edit-a-template-category}

Es ist derzeit nicht möglich, eine ganze Kategorie zu löschen oder Vorlagen stapelweise in eine neue Kategorie zu verschieben.

Sie können jedoch jede Vorlage innerhalb der Kategorie bearbeiten, um entweder einen vorhandenen Kategorienamen oder einen brandneuen Namen zu erstellen. Dadurch werden alle vorhandenen Vorlagen in einen anderen Ordner verschoben. Sobald der Ordner leer ist, wird er ausgeblendet.

## Zusammenführen und Deduplizieren von Kategorien {#merge-and-de-duplicate-categories}

Wenn Ihr [!DNL Sales Connect]-Konto auf die neueste Version von Vorlagen aktualisiert wird, werden Sie möglicherweise rote Warnhinweissymbole neben einigen Ihrer Vorlagenkategorien feststellen. Diese geben an, dass mehrere Kategorien mit demselben Namen vorhanden sind. Wenn Sie das Menü rechts neben Ihrer Kategorie öffnen, können Sie diese doppelten Kategorien mit einem Klick auf eine Schaltfläche zusammenführen.

## Löschen einer Vorlagenkategorie {#delete-a-template-category}

1. Klicken Sie auf das Menüsymbol neben einer Kategorie und dann auf **[!UICONTROL Löschen]**.

1. Klicken Sie **[!UICONTROL Löschen]** zur Bestätigung.

>[!NOTE]
>
>Kategorien können nicht gelöscht werden, wenn sie noch Vorlagen enthalten. Diese Vorlagen vor dem Löschen einer Kategorie verschieben/löschen
