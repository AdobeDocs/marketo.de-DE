# Bearbeiten einer Kategorie {#edit-a-template-category}

Derzeit ist es nicht möglich, eine ganze Kategorie zu löschen oder Vorlagen stapelweise in eine neue Kategorie zu verschieben.

Sie können jede Vorlage in der Kategorie entweder mit einem vorhandenen Kategorien- oder einem neuen Namen bearbeiten. Dadurch werden alle vorhandenen Vorlagen in einen anderen Ordner verschoben. Sobald der Ordner leer ist, wird er ausgeblendet.

Wir arbeiten daran, die Fähigkeit dazu in Massen hinzuzufügen.
