# Kategorien zusammenführen und deduplizieren {#merge-and-de-duplicate-categories}

Wenn Ihr Sales Connect-Konto auf die neueste Version von Vorlagen aktualisiert wird, werden Sie möglicherweise rote Warnsymbole neben einigen Ihrer Vorlagenkategorien bemerken. Diese geben an, dass mehrere Kategorien mit demselben Namen vorhanden sind. Wenn Sie das Menü rechts neben Ihrer Kategorie öffnen, können Sie diese doppelten Kategorien mit einem Klick auf eine Schaltfläche zusammenführen.
