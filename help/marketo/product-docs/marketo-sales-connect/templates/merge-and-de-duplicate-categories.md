# Zusammenführen und Deduplizieren von Kategorien {#merge-and-de-duplicate-categories}

Wenn Ihr [!DNL Sales Connect]-Konto auf die neueste Version von Vorlagen aktualisiert wird, werden Sie möglicherweise rote Warnhinweissymbole neben einigen Ihrer Vorlagenkategorien feststellen. Diese geben an, dass mehrere Kategorien mit demselben Namen vorhanden sind. Wenn Sie das Menü rechts neben Ihrer Kategorie öffnen, können Sie diese doppelten Kategorien mit einem Klick auf eine Schaltfläche zusammenführen.
