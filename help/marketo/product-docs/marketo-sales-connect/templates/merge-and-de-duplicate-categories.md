# Kategorien zusammenführen und deduplizieren {#merge-and-de-duplicate-categories}

Wenn Ihr Sales Connect-Konto auf die neueste Version von Vorlagen aktualisiert wird, werden möglicherweise neben einigen Ihrer Vorlagenkategorien rote Warnsymbole angezeigt. Diese geben an, dass Sie mehrere Kategorien mit demselben Namen haben. Durch Öffnen des Menüs rechts von der Kategorie können Sie diese duplizierten Kategorien mit einem Klick auf eine Schaltfläche zusammenführen.
