# Link-Landingpage abmelden {#unsubscribe-link-landing-page}

Wenn eine Person auf den Link zum Abbestellen klickt, wird sie zu einer Landingpage zum Abbestellen gebracht, in der sie auswählen kann, was sie abbestellen möchten und warum.

![](assets/1.jpg)

Diese Informationen werden in der Ansicht mit den Personendetails gespeichert, damit sie später angezeigt werden können.
