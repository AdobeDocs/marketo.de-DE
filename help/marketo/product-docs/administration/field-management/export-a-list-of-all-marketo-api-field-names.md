# Eine Liste aller Marketo-API-Feldnamen exportieren {#export-a-list-of-all-marketo-api-field-names}

Wenn Sie unsere SOAP-API oder Munchkin-API verwenden, benötigen Sie eine Liste aller Felder und deren API-Namen. So bekommt man diese Liste.

>[!NOTE]
>
>**Erforderliche Administratorberechtigungen**

1. Navigieren Sie zu **Admin** und klicken Sie auf **Feldverwaltung**.

   ![](assets/image2014-9-24-14-3a4-3a54.png)

1. Klicken **Feldnamen exportieren** , um die Tabelle herunterzuladen.

   ![](assets/image2014-9-24-14-3a5-3a6.png)

Gut! Sie haben jetzt eine Tabelle mit einer Liste aller Felder und deren API-Namen.

![](assets/image2014-9-24-14-3a5-3a19.png)

>[!NOTE]
>
>Die Zeichenbeschränkung für MLM-API-Namen beträgt 255.
