# Eine Branding-Domäne löschen {#delete-a-branding-domain}

1. Navigieren Sie zum **Admin**-Bereich.

   ![](assets/delete-a-branding-domain-1.png)

1. Klicken **Email**.

   ![](assets/delete-a-branding-domain-2.png)

1. Wählen Sie in der Tabelle Branding-Domänen die Domäne aus, die Sie entfernen möchten, und klicken Sie auf **Löschen**.

   ![](assets/delete-a-branding-domain-3.png)

   >[!NOTE]
   >
   >Wenn Sie die primäre Branding-Domäne löschen möchten, müssen Sie zunächst eine andere Branding-Domäne als die primäre Domäne auswählen.
