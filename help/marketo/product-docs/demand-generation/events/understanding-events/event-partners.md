# Ereignispartner {#event-partners}

Verbinden Sie Marketo mit einem Veranstaltungspartner, um Ihre Online-Ereignisse zu automatisieren. Wenn Sie sie verbinden, können Sie:

* Webinar-Ereignisse verwalten
* Push-Registrierungsinformationen, die auf einer Marketo-Landingpage erfasst werden, an den konfigurierten Ereignispartner senden
* Abrufen von Anwesenheitsinformationen automatisch in eine Marketo-Veranstaltung
* Erfassen Sie den Status der Personen als Mitglieder des Programms

Informationen hierzu finden Sie in der Anleitung zur Integration Ihrer Marketo-Online-Events mit Ihrem Veranstaltungspartner:
