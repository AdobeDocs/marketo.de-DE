# Berechtigungen {#permissions}

Es gibt fünf Standardprofile mit vordefinierten Berechtigungen, die Sie im Dynamic Chat bearbeiten können. Sie können auch ein benutzerdefiniertes Profil mit einem benutzerdefinierten Berechtigungssatz erstellen. Lasst uns über beide gehen.

## Bearbeiten vorhandener Berechtigungen {#edit-existing-permissions}

1. Adobe Admin Console Klicken Sie in der [](https://adminconsole.adobe.com/){target="_blank"} auf **Dynamic Chat**.

   ![](assets/permissions-1.png)

1. Wählen Sie auf **Registerkarte** das Profil aus, das Sie bearbeiten möchten. In diesem Beispiel wählen wir &quot;**&quot;**.

   ![](assets/permissions-2.png)

1. Klicken Sie auf **Registerkarte** Berechtigungen“.

   ![](assets/permissions-3.png)

1. Wählen Sie den Bereich des Profils aus, den Sie bearbeiten möchten. In diesem Beispiel wählen wir Live-Chat. Klicken Sie auf das Stiftsymbol.

   ![](assets/permissions-4.png)

1. Verfügbare Berechtigungselemente werden auf der linken Seite aufgeführt. Sie können Berechtigungen einzeln oder alle gleichzeitig hinzufügen. Klicken Sie auf das **+**.

   ![](assets/permissions-5.png)

   >[!NOTE]
   >
   >Durch die Aktivierung der automatischen Einbindung werden alle Berechtigungselemente zur eingeschlossenen Liste hinzugefügt. Wenn neue Berechtigungselemente verfügbar werden, werden sie automatisch für dieses Produktprofil eingefügt.

1. Klicken Sie auf **Speichern**.

   ![](assets/permissions-6.png)

Sie können diesen Vorgang nun für alle anderen Dynamic Chat-Bereiche wiederholen.

![](assets/permissions-7.png)

## Profil erstellen {#create-a-profile}

1. Adobe Admin Console Klicken Sie in der [](https://adminconsole.adobe.com/){target="_blank"} auf **Dynamic Chat**.

   ![](assets/permissions-8.png)

1. Klicken Sie auf **Registerkarte** Produktprofile“ auf **Neues Profil**.

   ![](assets/permissions-9.png)

1. **Name** Ihr Produktprofil. Optional können Sie ihm einen Anzeigenamen und/oder eine Beschreibung geben und festlegen, dass Benutzer benachrichtigt werden, wenn sie hinzugefügt/entfernt werden. Klicken Sie auf **Speichern**, wenn Sie fertig sind.

   ![](assets/permissions-10.png)

1. Ihr neues Profil wird auf der Registerkarte Produktprofile angezeigt. Wählen Sie es aus.

   ![](assets/permissions-11.png)

1. Führen Sie nun für jeden gewünschten Bereich die Schritte 3 [ 6 ](#edit-existing-permissions) oben beschriebenen Bereich aus.
