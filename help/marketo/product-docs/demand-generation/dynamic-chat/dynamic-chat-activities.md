# Dynamische Chat-Aktivitäten {#dynamic-chat-activities}

Der dynamische Chat bietet mehrere Filter und Trigger zur Verwendung in Ihren Smart-Listen.

![](assets/dynamic-chat-activities-1.png)

## Zu beachten {#things-to-note}

* Bedingungen werden in den Schritten zum dynamischen Chat-Fluss unterstützt
* Dynamische Chat-Aktivitäten können mit [Marketo Sales Insight](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/features/dynamic-chat-integration.md){target=&quot;_blank&quot;}
* Sie können einzelne Aktivitäten des dynamischen Cats im Aktivitätsprotokoll eines Personendatensatzes anzeigen
