# Einbetten eines Formulars auf Ihrer Website {#embed-a-form-on-your-website}

Mit Marketo können Sie unsere Formulare auf Ihrer eigenen Website einbetten. So erhalten Sie Zugriff auf den Einbettungscode.

1. Wechseln Sie zu **Marketingaktivitäten**.

   ![](assets/login-marketing-activities-4.png)

1. Wählen Sie das Formular aus.

   ![](assets/image2014-9-15-12-3a12-3a14.png)

1. Klicken Sie unter &quot;**Formulaaktionen**&quot;auf &quot;**Einbettungscode**&quot;.

   >[!NOTE]
   >
   >Das Formular muss genehmigt werden, damit das Element **Einbettungscode** sichtbar/nutzbar ist.

   ![](assets/image2014-9-15-12-3a12-3a20.png)

   >[!CAUTION]
   >
   >**[Das Vorfüllen des Formulars](/help/marketo/product-docs/administration/settings/edit-landing-page-settings.md)** funktioniert nicht, wenn Sie den Einbettungscode des Formulars auf Ihren eigenen Seiten _oder_ einer Marketo-Einstiegsseite verwenden. Das Vorbefüllen von Formularen ist nur dann sinnvoll, wenn das Formular in einer Marketo-Landingpage über die Option Element einfügen verwendet wird.

1. Wählen Sie den Einbettungscode aus/kopieren Sie ihn und klicken Sie dann auf **Schließen** .

   ![](assets/image2014-9-15-12-3a12-3a31.png)

>[!TIP]
>
>Sobald der Code auf Ihrer Website eingebettet ist, werden alle Änderungen am Formular in Marketo bei der Formulargenehmigung an Ihre Site gesendet. Sie müssen keine weiteren Änderungen am Code vornehmen.

Geben Sie nun einfach den Einbettungscode an Ihren Web-Entwickler weiter und lassen Sie ihn von diesen zu Ihrer Site hinzufügen.

>[!NOTE]
>
>Wenn Ihr Entwickler das Erscheinungsbild anpassen oder auf erweiterte API-Funktionen zugreifen möchte, zeigen Sie ihnen die [Forms 2.0-Entwicklerseite](https://experienceleague.adobe.com/en/docs/marketo-developer/marketo/javascriptapi/forms-api-reference) an.

Gut gemacht! Soll der Einbettungscode [ Lightbox-Code](/help/marketo/product-docs/demand-generation/forms/form-actions/use-a-form-in-a-lightbox.md) enthalten? Das ist auch einfach!
