# Vorlage 5e {#template-5e}

Diese Vorlage enthält den folgenden Inhalt:

* Eine Kopfzeile (optional)
* Ein primärer Abschnitt

* Drei Hauptteilabschnitte (optional)
* Fußzeile (optional)
