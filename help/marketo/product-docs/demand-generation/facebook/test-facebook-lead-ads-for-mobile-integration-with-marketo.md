# Testen von Facebook-Lead-Anzeigen für die mobile Integration mit Marketo {#test-facebook-lead-ads-for-mobile-integration-with-marketo}

Nachdem Sie Ihre Lead-Anzeige erstellt haben, sollten Sie sie testen!

>[!PREREQUISITES]
>
>Sie müssen [die Facebook-Lead-Anzeigen-Integration einrichten](/help/marketo/product-docs/demand-generation/facebook/set-up-facebook-lead-ads.md).

1. Wählen Sie im Facebook Power Editor eine Kampagne und eine Anzeige aus und klicken Sie auf **Bearbeiten**.

1. Klicken **unter** auf den Link **In App anzeigen**.

   ![](assets/image2016-5-13-15-3a2-3a38.png)

1. An das Facebook-Konto wird eine neue Benachrichtigung gesendet, auf die Sie mit dem autorisierten Konto auf dem Mobilgerät zugreifen können. Klicken Sie auf **OK**.

   ![](assets/image2016-3-11-8-3a35-3a7.png)

1. Facebook Tippen Sie auf Ihrem Mobilgerät in **Mobile App auf** Benachrichtigungen“.

   ![](assets/image2016-3-11-8-3a38-3a35.png)

1. Tippen Sie in den Benachrichtigungen auf **Ihre Anzeige ist für die Vorschau bereit**.

   ![](assets/image2016-3-11-8-3a41-3a59.png)

1. Senden Sie Ihre Test-Lead-Anzeigeneinheit, indem Sie auf Ihren Aktionsaufruf tippen und das von Ihnen erstellte Formular ausfüllen.

   ![](assets/image2016-3-11-8-3a52-3a20.png)

   >[!NOTE]
   >
   >Dies ist nur ein Beispiel, das einen Aufruf für weitere Informationen verwendet. Ihr Aktionsaufruf für die Lead-Anzeigeneinheit kann anders sein.

1. Hier passiert der Zauber! Nachdem Sie Ihr Formular übermittelt haben, [ Sie (eine Smart List in Marketo erstellen](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md) als Teil eines Programms oder in der Lead-Datenbank, die den Filter **Ausgefülltes Facebook-Lead-Anzeigen-Formular** verwendet. Fügen Sie den Namen des Lead-Werbeformulars für das soeben gesendete Formular ein.

   ![](assets/image2016-3-11-8-3a59-3a34.png)

1. Klicken Sie nun auf die Registerkarte Leads , um zu überprüfen, ob die Synchronisierung ordnungsgemäß funktioniert.

   ![](assets/image2016-3-11-15-3a27-3a54.png)
