# Smart-Listen {#understanding-smart-lists}

Mit intelligenten Listen können Sie mithilfe einfacher Filter bestimmte Personengruppen suchen. Sie werden überall in Marketo eingesetzt, z. B. in Smart-Kampagnen, Programmen, Berichten, Segmentierungen, Geschäftsmodellen, Interaktionsprogrammen, dynamischen Inhalten und mehr. Je nach Ihren Anforderungen können Smart-Listen an zwei Stellen in Marketo erstellt werden:

1. In der **Datenbank** sind sie für die gemeinsame/globale Verwendung verfügbar (z. B. &quot;Alle abgemeldeten Personen&quot;).
1. In **Programmen** als lokale Assets, in denen sie eine Gruppe von Personen finden, die für dieses spezifische Programm relevant sind (z. B. &quot;Teilnehmer an einer Handelsmesse&quot;).

Die wichtigsten Schritte zum Erstellen einer Smart-Liste sind:

1. [Erstellen einer Smart-Liste](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md){target="_blank"}
1. [Filter suchen und zu einer Smart-Liste hinzufügen](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/find-and-add-filters-to-a-smart-list.md){target="_blank"}
1. [Definieren von Smart-List-Filtern](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/define-smart-list-filters.md){target="_blank"}

Hier ist ein Beispiel dafür, wie eine Smart-Liste aussieht.

## Hauptregisterkarte {#main-tab}

Hier können Sie [den Namen der Smart-Liste bearbeiten](/help/marketo/product-docs/core-marketo-concepts/miscellaneous/rename-a-marketo-asset.md){target="_blank"} und Informationen zur Smart-Liste anzeigen.

![](assets/understanding-smart-lists-1.png)

## Registerkarte &quot;Personen&quot; {#people-tab}

Hier sehen Sie Ihre Ergebnisse, d. h. alle Personen, die sich für die Filter qualifizieren, die Sie [hinzugefügt und definiert haben](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/find-and-add-filters-to-a-smart-list.md){target="_blank"} auf der Registerkarte Smart-Liste hinzugefügt haben.

![](assets/understanding-smart-lists-2.png)

## Registerkarte &quot;Smart List&quot; {#smart-list-tab}

Sie können Filter hinzufügen und definieren, um eine bestimmte Gruppe von Personen auf der Registerkarte Personen anzuzeigen.

![](assets/understanding-smart-lists-3.png)
