# E-Mail-Vorschau {#previewing-an-email}

Klicken Sie mit der rechten Maustaste auf eine E-Mail-Karte, um sie in der Vorschau anzuzeigen, bevor Sie den Trigger abrufen.

1. Tippen Sie auf einer E-Mail-Karte auf das Menü mit den drei Punkten.

   ![](assets/image2015-9-25-11-3a30-3a52.png)

1. Tippen **Vorschau von E-Mail**.

   ![](assets/image2015-7-14-16-3a42-3a21.png)

1. Sie können die E-Mail auf Ihrem Gerät anzeigen.

   ![](assets/image2015-9-25-11-3a23-3a42.png)

   >[!NOTE]
   >
   >Um ein Muster direkt über die Seite &quot;E-Mail-Vorschau&quot;zu senden, tippen Sie oben rechts auf das Papierfliegersymbol.

   Ausgezeichnet!
