# CX Enterprise-Mitarbeiter - Übersicht {#overview}

CX Enterprise Coworker ist eine KI-native Marketing-Anwendung, mit der Sie innerhalb von Minuten von einer einzigen Eingabeaufforderung zu einer Kampagne für die Überprüfung bereit sind.

## Zugriff

>[!AVAILABILITY]
>
>CX Enterprise Coworker ist als kostenlose Testversion bis zum 30. September 2026 verfügbar.

1. Navigieren Sie zu coworker-essentials.experience.adobe.com.

1. Melden Sie sich mit Ihrer geschäftlichen E-Mail an oder erstellen Sie ein Konto.

   Bestehende Adobe-Kunden werden automatisch genehmigt und direkt zur Testumgebung weitergeleitet, um die Bedingungen zu akzeptieren.

   Neue Benutzer werden aufgefordert, ein Konto zu erstellen. Das Team von CX Enterprise Coworker prüft Anfragen und genehmigt geschäftliche E-Mails. Sie erhalten eine E-Mail, sobald Ihre Anfrage genehmigt wurde.

1. Akzeptieren Sie die Testbedingungen.

1. Richten Sie Ihre Marke unter **Ihr Material** > **Marken** ein. CX Enterprise Coworker verwendet dies als Standard für jede Kampagne, die Sie generieren.

1. (Optional) Laden Sie eine HTML-E-Mail-Vorlage unter **E-Mail-Vorlagen** hoch, damit CX Enterprise Coworker Ihr Design beim Generieren von Inhalten anwenden kann.

## Was Sie mit CX Enterprise Coworker tun können

CX Enterprise Coworker bringt fünf Workflows, die in der Regel fünf verschiedene Tools umfassen, in ein Gesprächserlebnis:

- **Verwandeln Sie einen Brief in einen Kampagnenplan**. Wenn Sie eine Eingabeaufforderung ablegen oder eine Zusammenfassung einfügen, gibt CX Enterprise Coworker einen strukturierten End-to-End-Plan zurück.
- **Erstellen und Verfeinern von Zielgruppen**. Laden Sie eine CSV-Datei hoch oder verbinden Sie Ihre Daten und verfeinern Sie dann das Segment in der Konversation.
- **Generieren von markeninternen Inhalten**. CX Enterprise Coworker schreibt E-Mails, die Ihrer Markensprache, Ihren Referenzen und Vorlagen entsprechen.
- **Entwerfen Sie mehrstufige Journey**. CX Enterprise Coworker stellt Pflegesequenzen zusammen, nicht nur einmalige Sendungen.
- **Testversand-E-Mails**. Zeigen Sie vor dem Start jede Nachricht in Ihrem eigenen Posteingang in der Vorschau an.
- **Starten und analysieren**. _(in Kürze verfügbar)_ Senden Sie Kampagnen und erhalten Sie Einblicke in die nächsten Schritte, ohne den CX Enterprise-Mitarbeiter verlassen zu müssen.

## Funktionsweise der KI

CX Enterprise Coworker folgt einer einfachen, mehrgängigen Gesprächsschleife:

**Eingabe**. Sie geben ein Kampagnenziel, Ihre Zielgruppe (hochgeladen oder verbunden) und den optionalen Markenkontext an.

**Verarbeitung**. Die KI von CX Enterprise Coworker agent generiert einen Kampagnenplan, erstellt eine Journey und entwirft personalisierte Inhalte.

**Ausgabe**. Ein Kampagnenentwurf wird auf Ihrem Campaign-Board angezeigt und kann überprüft, iteriert und schließlich gestartet werden.

Sie können jederzeit mit CX Enterprise Coworker sprechen (z. B. „die zweite E-Mail dringlicher gestalten“, „die Betreffzeilen verkürzen“ oder „die Zielgruppe in abgelaufene Kundinnen und Kunden aus dem 3. Quartal austauschen„) und die Artefakte aktualisieren.

>[!CAUTION]
>
>Die Ausgaben müssen vor dem Senden immer überprüft werden.

## Tipps für die ersten Schritte

Einige Dinge, die frühe Nutzer festgestellt haben, machen einen echten Unterschied:

- **Beginnen Sie mit einem bestimmten, echten Anwendungsfall**. „Win-Back-Serie für Kunden, die nicht in 90 Tagen bestellt haben“ schlägt „machen mich eine Kampagne.“ Siehe _Anwendungsfälle_ für spezifische Beispiele.
- **Bereinigen Sie Ihre Zielgruppendaten**. CX Enterprise Coworker kann nur so gut personalisieren, wie es Ihre CSV-Datei erlaubt. Schließen Sie alle Kontakte aus, die Sie vor dem Hochladen nicht per E-Mail versenden sollten.
- **Investieren Sie in Ihr Markensetup**. CX Enterprise Coworker extrahiert Markendetails, wenn Sie sich anmelden. Eine kurze Übersicht über **Ihre Inhalte** > **Marken** kommt jeder Kampagne zugute, die Sie generieren.
- **Testversand per E-Mail durchführen**. Senden Sie vor dem Start einen Testversand und überprüfen Sie ihn in Ihrem eigenen Posteingang.
- **Iterieren, nicht neu starten**. Betrachten Sie die erste Ausgabe von CX Enterprise Coworker als Ausgangspunkt. Der schnellste Weg zu großem Erfolg sind ein paar Runden Verfeinerung, keine neue Eingabeaufforderung.
- **Exportieren Sie , wenn Sie eine Papierspur benötigen**. Exportieren Sie einzelne E-Mails als HTML oder die gesamte Kampagne als Word-Dokument oder PDF zur Teamüberprüfung.

## Was in der Testversion verfügbar ist

CX Enterprise Coworker ist ein Produkt in aktiver Entwicklung. Hier finden Sie Informationen zum Einstieg:

- **Testfenster**: Jetzt bis zum 30. September 2026.
- **Annahme erforderlich**: Sie müssen die Testbedingungen überprüfen und akzeptieren, bevor Sie auf das Produkt zugreifen können.
- **Limit für den Versand**: Bis zu 5.000 E-Mails pro Monat.
- **Zielgruppenlimit**: Bis zu 10.000 Kontakte pro Kampagne.
- **Datenanleitung**: Laden Sie keine sensiblen oder regulierten Daten hoch. CX Enterprise Coworker ist derzeit nicht für HIPAA-regulierte Branchen vorgesehen.
- **Compliance**: Sie sind für die Einhaltung von E-Mail-Vorschriften verantwortlich (Abmelde-Links, Datenschutzrichtlinien usw.).

Während der Testphase werden neue Funktionen bereitgestellt. Ihr Feedback prägt das weitere Vorgehen. Senden Sie Feedback über den produktinternen Feedback-Link.
