# Planen Sie eine Smart-Batch-Kampagne in der Ansicht Programmzeitplan neu {#reschedule-a-batch-smart-campaign-in-the-program-schedule-view}

Möchten Sie die Ausführung einer Batch-Kampagne neu planen? Ziehen Sie sie in der Ansicht Programmplan per Drag-and-Drop in die Zukunft.

1. Wählen Sie die Smart-Kampagne aus, die Sie neu planen möchten.

   ![](assets/image2015-5-19-12-3a8-3a28.png)

1. Ziehen Sie per Drag-and-Drop auf das neue Datum, an dem die Kampagne ausgeführt werden soll.

![](assets/image2015-5-19-12-3a12-3a1.png)

1. Sehr gut! Die intelligente Kampagne wurde neu geplant. Die Uhrzeit kann auch über die Dropdown-Liste **[!UICONTROL Datum]** in den Eintragsdetails geändert werden.

   ![](assets/image2015-5-19-12-3a15-3a38.png)
