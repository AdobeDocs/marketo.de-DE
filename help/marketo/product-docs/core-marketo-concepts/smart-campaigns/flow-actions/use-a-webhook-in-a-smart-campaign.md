# Verwenden eines Webhooks in einer Smart-Kampagne {#use-a-webhook-in-a-smart-campaign}

So verwenden Sie [Webhook](https://developers.marketo.com/documentation/webhooks/), fügen Sie sie zu einem [Smart-Kampagne](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/add-a-flow-step-to-a-smart-campaign.md) als Fluss-Aktion.

>[!AVAILABILITY]
>
>Nicht alle Marketo Engage-Benutzer haben diese Funktion erworben. Weitere Informationen erhalten Sie vom Adobe Account Team (Ihrem Kundenbetreuer).

1. [Erstellen einer Smart-Kampagne](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/create-a-new-smart-campaign.md).

   >[!NOTE]
   >
   >Webhooks können nur in Trigger-Kampagnen verwendet werden.

1. Navigieren Sie zu **Fluss** Registerkarte und ziehen Sie die **Webhook aufrufen** Flussaktion.

   ![](assets/image2014-9-22-15-3a8-3a2.png)

1. Wählen Sie den Webhook aus.

   ![](assets/image2014-9-22-15-3a8-3a5.png)

1. Sie können Webhooks auch in einer intelligenten Liste verwenden.

   ![](assets/2017-05-02-10-54-38.png)

1. Schließlich können Sie Webhooks in **Auswahl hinzufügen** in einem Flussschritt.

   ![](assets/image2014-9-22-15-3a8-3a13.png)
