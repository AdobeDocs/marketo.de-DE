# Durchführen einer Batch-Smart-Kampagne jetzt | Registerkarte &quot;Plan&quot; {#run-a-batch-smart-campaign-now-schedule-tab}

Nachdem Sie die Erstellung Ihrer Batch-Kampagne abgeschlossen haben, können Sie sie jetzt oder später ausführen. So kann man es sofort ausführen.

1. Wählen Sie die Batch-Kampagne aus, wechseln Sie zur Registerkarte **[!UICONTROL Plan]** und klicken Sie auf **[!UICONTROL Einmal ausführen]**.

   ![](assets/run-a-batch-smart-campaign-now-schedule-tab-1.png)

1. Stellen Sie sicher, dass **[!UICONTROL Jetzt ausführen]** ausgewählt ist, und klicken Sie auf **[!UICONTROL Ausführen]**.

   ![](assets/run-a-batch-smart-campaign-now-schedule-tab-2.png)

1. Bestätigen Sie dies, indem Sie erneut auf **[!UICONTROL Ausführen]** klicken.

   ![](assets/run-a-batch-smart-campaign-now-schedule-tab-3.png)

   Einfach pitasie! Sie können auch [die Ausführung für später planen](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/using-smart-campaigns/schedule-a-batch-smart-campaign-to-run-later.md){target="_blank"}, wenn Sie es bevorzugen.
