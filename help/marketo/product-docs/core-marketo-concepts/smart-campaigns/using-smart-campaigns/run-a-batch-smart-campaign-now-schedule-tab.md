# Durchführen einer Batch-Smart-Kampagne jetzt | Registerkarte &quot;Planung&quot; {#run-a-batch-smart-campaign-now-schedule-tab}

Nachdem Sie die Erstellung Ihrer Batch-Kampagne abgeschlossen haben, können Sie sie jetzt oder später ausführen. So kann man es sofort ausführen.

1. Wählen Sie die Batch-Kampagne aus und navigieren Sie zum **[!UICONTROL Zeitplan]** und klicken Sie auf **[!UICONTROL Einmal ausführen]**.

   ![](assets/run-a-batch-smart-campaign-now-schedule-tab-1.png)

1. Stellen Sie sicher **[!UICONTROL Jetzt ausführen]** ausgewählt ist und auf **[!UICONTROL Ausführen]**.

   ![](assets/run-a-batch-smart-campaign-now-schedule-tab-2.png)

1. Bestätigen durch Klicken auf **[!UICONTROL Ausführen]** einmalig.

   ![](assets/run-a-batch-smart-campaign-now-schedule-tab-3.png)

   Einfach pitasie! Sie können auch [planen läuft für später](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/using-smart-campaigns/schedule-a-batch-smart-campaign-to-run-later.md){target="_blank"} wenn Sie es vorziehen.
