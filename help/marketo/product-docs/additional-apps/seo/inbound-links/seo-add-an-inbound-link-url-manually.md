# SEO - Manuelles Hinzufügen einer URL für eingehende Links {#seo-add-an-inbound-link-url-manually}

Sie können Eingehende Links manuell hinzufügen. So geht&#39;s:

1. Wechseln Sie zum Abschnitt **[!UICONTROL Eingehende Links]** .

   ![](assets/image2014-9-18-13-3a40-3a3.png)

1. Klicken Sie auf **[!UICONTROL URLs hinzufügen]**.

   ![](assets/image2014-9-18-13-3a40-3a8.png)

1. Geben Sie die URL ein. Klicken Sie auf **[!UICONTROL Hinzufügen]**.

   ![](assets/image2014-9-18-13-3a40-3a32.png)

   >[!TIP]
   >
   >Wusstest du, dass du deinen eingehenden Link [zu einer neuen oder vorhandenen Liste hinzufügen kannst](/help/marketo/product-docs/additional-apps/seo/understanding-seo/seo-managing-lists.md){target="_blank"}? Schau es dir an!

   ![](assets/image2014-9-18-13-3a41-3a14.png)

Fick dich an! Ihre neue URL sollte nun der Liste hinzugefügt werden. Es kann einige Zeit dauern, bis alle Daten erfasst werden.
