# SEO - Hinzufügen/Entfernen einer URL für eingehende Links aus einer Liste {#seo-add-remove-an-inbound-link-url-from-a-list}

Sie können Ihre URLs für eingehende Links nach Herzenslust organisieren.

## Hinzufügen einer eingehenden Link-URL zu einer Liste {#add-an-inbound-link-url-to-a-list}

1. Navigieren Sie zum Abschnitt **[!UICONTROL Eingehende Links]** .

   ![](assets/image2014-11-20-18-3a27-3a27.png)

1. Bewegen Sie den Mauszeiger über die URL des eingehenden Links, den Sie kategorisieren möchten. Klicken Sie **[!UICONTROL Hinzufügen/Entfernen aus der Liste]**.

   ![](assets/image2014-11-20-18-3a27-3a40.png)

1. Klicken Sie auf die Liste, in der die URL für eingehende Links angezeigt werden soll.

   ![](assets/image2014-11-20-18-3a28-3a18.png)

>[!TIP]
>
>Sie können auch eine neue Liste für Ihr Keyword erstellen. Geben Sie einfach Ihren gewünschten Namen in [!UICONTROL Neue Liste erstellen] ein.

## Entfernen einer eingehenden Link-URL aus einer Liste {#remove-an-inbound-link-url-from-a-list}

Manchmal empfiehlt es sich, URLs für eingehende Links aus einer Liste zu entfernen.

1. Klicken Sie auf **[!UICONTROL Eingehende Links]**.

   ![](assets/image2014-11-20-18-3a28-3a41.png)

1. Klicken Sie auf **[!UICONTROL Dropdown-]** „Filtern nach“. Klicken Sie auf die Liste, die Sie bereinigen möchten.

   ![](assets/image2014-11-20-18-3a28-3a57.png)

1. Bewegen Sie den Mauszeiger über die URL des eingehenden Links, den Sie entfernen möchten. Klicken Sie **[!UICONTROL Hinzufügen/Entfernen aus der Liste]**.

   ![](assets/image2014-11-20-18-3a29-3a56.png)

1. Die Liste, die Sie organisieren möchten, wird überprüft. Klicken Sie auf den Listennamen, um die URL des eingehenden Links aus dieser Liste zu entfernen.

   ![](assets/image2014-11-20-18-3a30-3a10.png)

Du hast es geschafft! Aktualisieren Sie die Seite, um die Anzeige zu aktualisieren.
